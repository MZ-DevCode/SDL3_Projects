A simple survival game developed in **C++** using the **SDL3** library.

## Features
- **Delta Time**: Smooth movement regardless of frame rate.
- **Basic AI**: The enemy follows the player.
- **Score System**: Tracks current survival time and high score.
- **Debug Text**: Real-time stats displayed using SDL3 debug rendering.

## Controls
- **Arrows**: Move the player (Red square)
- **Goal**: Avoid the enemy (Blue square) as long as possible!
